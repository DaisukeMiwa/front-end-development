# 総合 Web コーディング入門ガイド

HTML / CSS / JavaScript / jQuery / GSAP を 1 ページで通読できる総合教材です。

## 公開ページ

GitHub Pages: https://daisukemiwa.github.io/front-end-development/

## ファイル構成

- `index.html` — 教材本体（単一HTMLファイルで完結）

## ローカル確認

```bash
# このフォルダで簡易サーバーを起動
python3 -m http.server 8000
# → http://localhost:8000/ をブラウザで開く
```

または `index.html` をブラウザで直接開いてもOKです（外部リソースに依存しないため）。
